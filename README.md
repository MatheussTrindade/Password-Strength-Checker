# 🔐 Password Strength Checker

## 📌 Overview
Ferramenta para analisar a força de senhas com base em critérios de segurança.

## 🚀 Features
- Verificação de tamanho
- Complexidade (maiúsculas, números, símbolos)
- Detecção de padrões comuns
- Score de segurança

## 🧠 Security Concepts
- Brute Force Attack
- Dictionary Attack
- Entropy (conceito básico)

## 🛠️ Tech Stack
- Python
